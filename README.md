## 📖 About The Project
CCJG Cleaning Services is a high-performance web platform designed to connect Canadian businesses and homeowners with top-tier cleaning professionals. 

In the cleaning industry, trust and professionalism are everything. This digital presence was engineered from the ground up to reflect those exact values: it is **blazing fast, visually immaculate, and highly accessible.**

🔗 **Live Platform:** [ccjg-cleaning-services.ca](https://ccjg-cleaning-services.ca/)

---

## ✨ Key Features

### 🏢 Business-Facing
- **Service Showcases:** Dedicated landing pages for Commercial, Residential, and Post-Construction cleaning.
- **Conversion-Optimized:** Strategic Call-to-Action (CTA) placements for fast quote requests.
- **Trust Signals:** Integrated testimonials, trust badges, and clear service area maps (Timmins/Ontario).
- **Mobile-First Experience:** 100% responsive design for clients booking on the go.

### 💻 Technical Architecture
- **Zero-JavaScript Frontend:** Powered by Astro's island architecture for instant load times.
- **SEO Masterclass:** Auto-generated sitemaps, perfect semantic HTML, and dynamic meta tags for local Canadian SEO domination.
- **Modern Styling:** Tailwind CSS utilized for a clean, consistent, and maintainable utility-first design system.

---

## 🛠️ The Tech Stack
This project leverages a cutting-edge modern web stack:

- **Framework:** [Astro](https://astro.build/) - For unmatched static site generation speed.
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) - For pixel-perfect, responsive UI components.
- **Language:** [TypeScript](https://www.typescriptlang.org/) - Ensuring type safety and robust code quality.
- **Deployment & Hosting:** [Hostinger](https://www.hostinger.com/) - Optimized Node.js environment with SSL encryption.

## 📂 Project Structure
A glimpse into the organized, scalable architecture of the platform:

```text
/ccjg-web
├── public/                 # Static assets (Favicons, optimized images)

│   └── styles/             # Global CSS and Tailwind directives

├── tailwind.config.cjs     # Tailwind theme, brand colors, and plugins
